# A Layout-Aware Resume Parsing Pipeline Using YOLO-OCR-NLP

## ภาพรวมโครงการ (Project Overview)
โครงการนี้มีเป้าหมายเพื่อพัฒนาระบบ Resume Parsing แบบ Layout‑Aware ซึ่งสามารถแปลงเรซูเม่ที่อยู่ในรูปแบบภาพหรือ PDF ซึ่งเป็นเอกสารที่ไม่มีโครงสร้าง (unstructured documents) ให้กลายเป็นข้อมูลเชิงโครงสร้างโดยอัตโนมัติ ระบบถูกออกแบบให้เข้าใจทั้ง 
โครงสร้างเชิงพื้นที่ของเอกสาร (document layout) และความหมายของข้อความ (semantic content) ร่วมกัน เพื่อเพิ่มความแม่นยำในการดึงข้อมูลจากเรซูเม่ที่มีรูปแบบหลากหลาย ระบบใช้แนวคิดแบบ Hybrid Pipeline โดยผสานเทคนิคจาก Computer Vision และ Natural Language Processing 
เข้าด้วยกัน เริ่มจากการใช้โมเดลตรวจจับวัตถุ (YOLO) เพื่อระบุและแยกส่วนประกอบของเรซูเม่ เช่น Education, Experience และ Skills จากนั้นใช้ Optical Character Recognition (OCR) เพื่อแปลงข้อความจากแต่ละส่วนออกมาเป็นตัวอักษร และใช้โมเดลประมวลผลภาษาธรรมชาติในการดึงข้อมูลเชิงความหมาย
จากข้อความภายในแต่ละ section ผลลัพธ์ของระบบคือข้อมูลเรซูเม่ในรูปแบบเชิงโครงสร้าง (Structured JSON) ซึ่งสามารถนำไปใช้งานต่อในระบบอื่นได้ โครงการนี้มุ่งเน้นการศึกษาและพัฒนากระบวนการ Resume Parsing ให้สอดคล้องกับงานวิจัยต้นแบบที่เน้นการผสานข้อมูลเชิงพื้นที่และเชิงความหมาย โดยจากผลลัพธ์ที่ได้
อาจจะนำไปต่อยอดต่อไป

##### โครงการนี้พัฒนาขึ้นโดยได้รับแรงบันดาลใจจากแนวคิดและกรอบงานของงานวิจัย “A Hybrid OCR–XGBoost–Transformer Pipeline for Resume Parsing with Spatial‑Semantic Integration”

## นิยามปัญหา (Problem Statement)
เรซูเม่ (Resume/CV) มักอยู่ในรูปแบบเอกสารที่ไม่มีโครงสร้าง เช่น PDF หรือรูปภาพ และมีรูปแบบการจัดวางที่หลากหลาย ทำให้การดึงข้อมูลสำคัญออกมาใช้งานโดยอัตโนมัติเป็นเรื่องยาก ระบบที่อาศัยการประมวลผลข้อความเพียงอย่างเดียวหรือกฎแบบตายตัวมักไม่สามารถรองรับโครงสร้างเรซูเม่ที่แตกต่างกัน
ได้อย่างมีประสิทธิภาพ โครงการนี้มีเป้าหมายเพื่อแก้ไขปัญหาดังกล่าวโดยพัฒนาระบบ Resume Parsing ที่เข้าใจทั้งโครงสร้างเชิงพื้นที่ของเอกสารและความหมายของข้อความ เพื่อแปลงเรซูเม่ที่ไม่มีโครงสร้างให้เป็นข้อมูลเชิงโครงสร้างได้อย่างถูกต้องและยืดหยุ่น

---

## วัตถุประสงค์ของโครงการ (Objectives)
- เพื่อพัฒนาระบบ Resume Parsing แบบ Layout‑Aware ที่สามารถเข้าใจโครงสร้างการจัดวางของเรซูเม่จากภาพหรือไฟล์ PDF
- เพื่อผสานเทคนิคจาก Computer Vision, OCR และ Natural Language Processing ในรูปแบบ Hybrid Pipeline
- เพื่อแยกส่วนข้อมูลหลักของเรซูเม่ (เช่น Education, Experience, Skills) โดยอัตโนมัติ
- เพื่อดึงข้อมูลเชิงความหมายจากแต่ละส่วนของเรซูเม่ได้อย่างเป็นระบบ
- เพื่อแปลงเรซูเม่ที่ไม่มีโครงสร้างให้เป็นข้อมูลเชิงโครงสร้าง (Structured JSON) ที่พร้อมนำไปใช้งานต่อ

---

## รายละเอียดชุดข้อมูล (Dataset Description)
ชุดข้อมูลที่ใช้ในโครงการนี้ประกอบด้วย เรซูเม่ (CV) จำนวน 200 ใบ ซึ่งอยู่ในรูปแบบไฟล์ภาพและ PDF โดยเรซูเม่แต่ละใบมีรูปแบบการจัดวาง (layout) ที่หลากหลาย เช่น แบบหนึ่งคอลัมน์และสองคอลัมน์ รวมถึงมีการใช้ทั้งภาษาไทยและภาษาอังกฤษ
ข้อมูลดังกล่าวถูกนำมาใช้เพื่อทดสอบและประเมินความสามารถของระบบในการทำความเข้าใจโครงสร้างเอกสารและดึงข้อมูลจากเรซูเม่ที่ไม่มีโครงสร้าง เรซูเม่ในชุดข้อมูลถูกนำมาใช้สำหรับการฝึกและทดสอบโมเดลตรวจจับส่วนประกอบของเอกสาร (layout detection) 
รวมถึงการดึงข้อความและข้อมูลเชิงความหมายจากแต่ละส่วน เช่น Education, Experience และ Skills เพื่อแปลงข้อมูลให้อยู่ในรูปแบบเชิงโครงสร้างตามวัตถุประสงค์ของโครงการ

## เครื่องมือและไลบรารี (Tools & Libraries)
- **YOLO (You Only Look Once**) ใช้สำหรับตรวจจับโครงสร้างและส่วนประกอบของเรซูเม่ (Layout / Section Detection) เช่น Education, Experience และ Skills จากภาพหรือ PDF
- **OpenCV** ใช้สำหรับการประมวลผลภาพเบื้องต้น เช่น การตัดภาพ (cropping) และการจัดการภาพก่อนส่งต่อให้ OCR
- **OCR Engine** (เช่น EasyOCR หรือ Tesseract OCR) ใช้สำหรับแปลงข้อความจากภาพในแต่ละ section ให้เป็นตัวอักษร (text extraction) โดยรองรับทั้งภาษาไทยและภาษาอังกฤษ
- **spaCy** ใช้สำหรับการประมวลผลภาษาธรรมชาติ (Natural Language Processing) และช่วยในการดึงข้อมูลเชิงความหมายจากข้อความในแต่ละ section
- **Python** ใช้เป็นภาษาในการพัฒนาและเชื่อมต่อโมดูลต่าง ๆ ภายในระบบ
- **JSON** ใช้เป็นรูปแบบข้อมูลผลลัพธ์ (Structured Output) สำหรับจัดเก็บข้อมูลเรซูเม่หลังผ่านกระบวนการ parsing
